# SZUD_vebsajt

Lični portfolio web sajt napravljen sa HTML, CSS i JavaScript kao deo školskog projekta. Sajt prikazuje moje veštine, projekte i kontakt informacije, sa responzivnim dizajnom i CSS animacijama.

## Struktura projekta

```
SZUD_vebsajt/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## Tehnologije

- HTML5
- CSS3 (animacije, CSS varijable, Grid, Flexbox)
- JavaScript (Vanilla JS, Intersection Observer API)

## Pokretanje

Otvoriti `index.html` u bilo kom modernom web pretraživaču.
